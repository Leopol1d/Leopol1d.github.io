Shortest Path Problem

迪杰斯特拉

解析

  1. 开一个dis数组，记录从给定点x到其他点的最短距离，初始化为max，；开一个visited数组，记录顶点是否被访问过
  2. 将x的dis初始为0
  3. 开始遍历，首先找到未被访问且距离x最短的顶点y，将y的visited赋值true
  4. 更新的dis中顶点z的距离：未被访问且 z与x的距离大于 x到y + y到z的距离
